클래스 선택자가 뭐예요?
클래스 선택자는 CSS에서 정말 중요한 개념이에요. 우리가 웹 페이지를 만들 때, 같은 종류의 여러 요소에 동일한 스타일을 적용하고 싶을 때 사용해요. 예를 들어, 버튼 여러 개에 같은 색깔, 글꼴, 크기를 주고 싶다면, 클래스를 사용해서 한 번에 스타일을 적용할 수 있어요.

실제 예제 보기
우리의 예제에서, <button> 요소에 class="instagram-button"이라고 붙였어요.

<button class="instagram-button"> 팔로우 </button>
<button class="instagram-button"> 메시지 보내기</button>
이런 식으로 클래스를 부여하면, 우리는 CSS에서 .instagram-button이라고 적고, 그 안에 스타일을 정의할 수 있어요.

CSS에서 클래스 사용하기
예를 들어, .instagram-button 클래스를 가진 모든 버튼에 파란색 배경과 흰색 글씨를 주고 싶다면, CSS에서 이렇게 적을 수 있어요:

.instagram-button {
  background-color: blue;
  color: white;
}
이렇게 하면 지금부터 instagram-button이라는 클래스를 가진 모든 요소가 파란색 배경과 흰색 글씨로 표현된답니다.

클래스 선택자의 장점
클래스 선택자의 가장 큰 장점은 재사용성이에요. 한번 정의해둔 스타일을 웹 페이지의 여러 곳에서 다시 쓸 수 있어요. 이건 코드를 더 깔끔하고 효율적으로 만들어줘요.
