HTML 기본 구조와 DOCTYPE 선언
우리가 웹 페이지를 만들기 위해서는 먼저, HTML 문서의 기본 구조를 알아야 해요. 그 시작점이 바로 <!DOCTYPE html>이에요. 이것은 우리가 만들 페이지가 HTML5 표준을 따른다는 것을 브라우저에 알려줘요. 이렇게 함으로써, 우리 페이지가 다양한 브라우저에서도 일관되게 보이도록 도와준답니다.

<html>, <head>, <body> 태그의 마법
이제, HTML 문서의 세 가지 주요 부분을 살펴볼 거예요.

<html> 태그: 이 태그는 전체 웹 페이지의 루트(root) 요소예요. 페이지에 있는 모든 다른 요소들은 이 안에 포함되어야 해요.
<head> 태그: 여기서는 페이지의 제목이나 스타일과 같은 메타데이터를 정의해요. 예를 들어, <title>My First Instagram</title> 처럼요.
<body> 태그: 페이지의 실제 내용이 들어가는 곳이에요. 예를 들어, 이미지, 텍스트, 버튼 등이 모두 여기에 위치하죠.
왜 이 구조를 사용해야 할까요?
이 구조를 사용하는 이유는 무엇일까요? 바로, 이렇게 함으로써 우리의 코드가 더 잘 조직되고, 브라우저가 페이지를 더 잘 이해하며 렌더링할 수 있기 때문이에요. 또한, 이 구조를 사용하면 나중에 페이지를 업데이트하거나 수정할 때 훨씬 더 쉬워진답니다.

실제 적용하기
그럼, 우리가 배운 것을 현재 코드에 어떻게 적용했는지 살펴볼까요? <DOCTYPE html> 선언으로 시작해, <html> 태그로 전체를 감싸고, <head>에는 우리 페이지의 제목과 스타일을 넣었어요. 그리고 <body>에는 우리의 실제 내용—이미지, 사용자 이름, 버튼등이 위치해 있어요.
