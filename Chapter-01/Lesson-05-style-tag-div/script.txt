안녕하세요, 친구들! 오늘은 정말 중요한 것에 대해 배울 거예요. 바로 웹 페이지를 예쁘고 정돈되게 만드는 방법이에요. 우리가 어떻게 HTML과 CSS를 사용하여 멋진 인스타그램 프로필 섹션을 만들 수 있는지 함께 살펴보아요. 준비됐나요? 그럼 시작해볼까요?

HTML 구조화 이해하기
우리가 웹 페이지를 만들 때, 텍스트 또는 이미지를 바로 넣지만, 이것들을 좀 더 잘 정리해야 할 때가 있어요. 예를 들어, ">wecode"라는 사용자 이름을 <div> 태그로 감싸는 것이죠. 이렇게 하면 나중에 CSS로 스타일을 적용할 때 훨씬 쉬워진답니다!

<div>
  >wecode
</div>
CSS로 스타일링 배우기
이제 스타일의 마법을 더해볼 차례예요. 우리는 <style> 태그를 사용하여 HTML 문서 상단에 CSS를 작성할 거예요. 이 방법으로 우리는 이미지나 텍스트에 다양한 스타일을 적용할 수 있어요.

<style>
  img {
    width: 77px;
    height: 77px;
    border-radius: 50%;
  }
  div {
    font-family: Arial;
    font-size: 20px;
  }
</style>
이렇게 해서 우리의 이미지는 아름답게 동그랗게 되고, 텍스트는 눈에 띄는 아리얼 글꼴로 20px 크기로 보이게 됐어요. 정말 멋지죠?

CSS 선택자의 힘
CSS에서는 선택자(selector)를 사용해 특정 HTML 요소들에게 스타일을 적용해요. 이 예제에서는 img와 div 요소를 타겟팅해서 스타일을 적용했어요. 이 방법으로 예쁘고 통일된 디자인을 만들 수 있답니다.

웹 페이지의 전반적인 외관 향상
마지막으로, 핵심을 이해해야 해요. CSS 스타일링과 HTML 구조화를 통해 우리는 사용자 경험을 크게 개선할 수 있어요. 이 모든 것이 웹 페이지를 더 예쁘고, 사용하기 쉽고, 관리하기 쉽게 만들어준답니다.
